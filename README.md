# SBI Bank Management System

## Project Description
The SBI Bank Management System is a Java-based console application developed to manage basic banking operations.  
It allows users to perform common banking tasks in a simple and efficient way using object-oriented programming concepts.

## Features
- Create new bank account
- Deposit amount
- Withdraw amount
- Check account balance
- View customer details
- Menu-driven console application

## Technologies Used
- Java
- OOPS Concepts
- Scanner class for user input

## Project Structure
- Main.java – Contains the main menu and program execution
- BankAccount.java – Handles account details and banking operations

## How to Run the Project
1. Clone or download the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run the `Main.java` file
4. Follow the menu options displayed in the console
